# 基于注解的缓存装饰器

<ruby>装饰器模式<rt>decorator pattern</rt></ruby>是一种非常常用的高级设计模式。

请实现一个基于`@Cache`注解的装饰器，能够将传入的服务类的`Class`进行装饰，使之具有缓存功能。

在提交Pull Request之前，你应当在本地确保所有代码已经编译通过，并且通过了测试(`mvn clean verify`)
