要完成这个挑战，你需要：

- 自己从零开始配置MyBatis。这会让你熟悉JDBC、ORM以及MyBatis的各种概念。
- 完成各种复杂的SQL查询方法。我们鼓励你尝试不同的XML或者注解方式。
